# About

Mesh Painting is a runtime Unreal Engine plugin for painting static and skeletal meshes during gameplay.

It uses GPU render target painting, UV island clipping, brush preview, color picking, eraser support, and multiplayer paint command replication.

## Core Features

- Runtime GPU brush painting
- Static and skeletal mesh support
- UV island clipping for large brushes
- Brush color, opacity, hardness, metallic, and roughness controls
- Color picker and eraser workflow
- Multiplayer replication with late join replay
- Shipping build focused runtime module
