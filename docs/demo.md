## What To Test

- Paint directly on the demo character in a packaged Windows build.
- Check the runtime brush preview and the final painted result on the mesh.
- Compare the lighter build against the Lumen build if you want to test performance or visual quality.

## Notes

The demo is hosted on Google Drive. If the file receives too many downloads in a short time, Google may temporarily limit access.

Windows or your browser may show an unknown publisher or potentially unsafe file warning because the demo is a downloadable executable from Google Drive. This warning is controlled by Windows, browser reputation, and code signing status, so it cannot be fully removed from the website side. Only run the demo if you trust the download source.
